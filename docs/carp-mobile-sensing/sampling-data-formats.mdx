title: Sampling Data Formats
description: Overview of data output formats used by CAMS
icon: file-code
---

Data format support depends on the selected data endpoint and optional transformer schema.

For detailed, up-to-date mappings between measure types and collected data classes, see:

- [Measure Types](/carp-mobile-sensing/measure-types)

When a transformer schema is configured (for example OMH), CAMS converts supported native CARP data models to that namespace during data handling.

If no transformer exists for a specific type, CAMS keeps the native CARP format for that measurement.
