# Health Class API

Complete reference for the `Health` class - the main interface for health data operations.

## Constructor

```dart
Health()
```

Creates a new `Health` instance. Typically used as a singleton.

```dart
final health = Health();
```

## Permission Methods

### hasPermissions

Check if permissions have been granted.

```dart
Future<bool?> hasPermissions(
  List<HealthDataType> types, {
  List<HealthDataAccess>? permissions,
})
```

**Returns**: `true` (granted), `false` (denied), or `null` (not determined)

[See Permissions docs](permissions)

### requestAuthorization

Request user authorization for health data access.

```dart
Future<bool> requestAuthorization(
  List<HealthDataType> types, {
  List<HealthDataAccess>? permissions,
})
```

**Returns**: `true` if authorization granted

[See Permissions docs](permissions)

### revokePermissions

Revoke previously granted permissions.

```dart
Future<bool> revokePermissions()
```

**Returns**: `true` if permissions revoked successfully

[See Permissions docs](permissions)

## Reading Methods

### getHealthDataFromTypes

Read health data points for specified types and time range.

```dart
Future<List<HealthDataPoint>> getHealthDataFromTypes({
  required List<HealthDataType> types,
  required DateTime startDate,
  required DateTime endDate,
  bool includeManualEntry = true,
})
```

[See Basic Reading docs](read-basic)

### getHealthIntervalDataFromTypes

Read health data aggregated into time intervals.

```dart
Future<List<HealthDataPoint>> getHealthIntervalDataFromTypes({
  required DateTime startDate,
  required DateTime endDate,
  required List<HealthDataType> types,
  required Duration interval,
  bool includeManualEntry = true,
})
```

[See Interval Reading docs](read-interval)

### getHealthAggregateDataFromTypes

Read pre-aggregated health data (primarily for workouts).

```dart
Future<List<HealthDataPoint>> getHealthAggregateDataFromTypes({
  required List<HealthDataType> types,
  required DateTime startDate,
  required DateTime endDate,
  int activitySegmentDuration = 1,
  bool includeManualEntry = true,
})
```

[See Aggregate Reading docs](read-aggregate)

### getTotalStepsInInterval

Get total step count for a time period.

```dart
Future<int?> getTotalStepsInInterval(
  DateTime startDate,
  DateTime endDate,
)
```

**Returns**: Total steps or `null` if no data

[See Steps Reading docs](read-steps)

## Writing Methods

### writeHealthData

Write numeric health data.

```dart
Future<bool> writeHealthData({
  required double value,
  required HealthDataType type,
  required DateTime startTime,
  DateTime? endTime,
  String? sourceId,
  String? sourceName,
  DateTime? recordingTime,
})
```

[See Basic Writing docs](write-basic)

### writeWorkoutData

Write workout/exercise session data.

```dart
Future<bool> writeWorkoutData({
  required HealthWorkoutActivityType activityType,
  required DateTime start,
  required DateTime end,
  double? totalEnergyBurned,
  int? totalEnergyBurnedUnit,
  double? totalDistance,
  int? totalDistanceUnit,
})
```

[See Workout Writing docs](write-workouts)

### writeBloodPressure

Write blood pressure readings.

```dart
Future<bool> writeBloodPressure({
  required double systolic,
  required double diastolic,
  required DateTime startTime,
  DateTime? endTime,
})
```

[See Blood Pressure docs](write-blood)

### writeBloodOxygen

Write blood oxygen saturation data.

```dart
Future<bool> writeBloodOxygen({
  required double saturation,
  required DateTime startTime,
  DateTime? endTime,
  double? flowRate,
})
```

[See Blood Oxygen docs](write-blood)

### writeMeal

Write nutrition/meal data.

```dart
Future<bool> writeMeal({
  required DateTime startTime,
  DateTime? endTime,
  MealType? mealType,
  String? name,
  double? calories,
  double? protein,
  double? carbs,
  double? fat,
  // ... many more micronutrient parameters
})
```

[See Nutrition docs](write-nutrition)

### writeMenstruationFlow

Write menstrual flow data.


[See Specialized Writing docs](write-specialized)
